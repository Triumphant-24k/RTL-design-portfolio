# Communication Protocols using Verilog HDL

This repository contains digital communication protocol designs implemented in Verilog HDL and verified using ModelSim simulation.

The objective of these projects is to understand how data is transmitted, received, synchronized, and processed in digital systems and SoCs.

---

## Tools Used

- Verilog HDL
- ModelSim
- Vivado (for synthesis and FPGA implementation)
- Git & GitHub

---

## Projects

### UART Transmitter
Status: ✅ Completed

Features:
- FSM-based design
- Start bit generation
- 8-bit serial data transmission
- Stop bit generation
- Baud rate control
- Busy flag indication
- Testbench verification

Concepts Learned:
- Finite State Machines (FSM)
- Data Serialization
- Clock Division
- Protocol Timing
- RTL Verification

---

### UART Receiver
Status: 🚧 In Progress
