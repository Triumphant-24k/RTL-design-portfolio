# RTL Design Portfolio

A collection of Verilog HDL projects developed as part of my journey toward becoming an RTL Design Engineer.

This repository contains digital design projects implemented in Verilog, simulated using ModelSim and Vivado ML Edition. The projects cover fundamental and intermediate concepts including combinational logic, sequential logic, finite state machines (FSMs), arithmetic circuits, counters, shift registers, and communication protocols.
